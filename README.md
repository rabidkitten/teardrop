This repo contains a UI components for React that displays a tear drop.This repo
is open source and will remain so.

![Sample](sample.png)

## Installation

```bash
npm i @ditus/teardrop
```

## Usage

```html
  <div
    style={{
      width: '400px',
      height: '400px',
    }}
  >
    <Teardrop {...args} />
  </div>
```

## Demo

To view the components, type:

```bash
npm i
npm run storybook
```
